Create boards and discussions on the community page

Last updated on 08/04/2020
Written by Kevin Legros
Updated over a week ago

You can :

  • Create a Board for a partner

  • Create a Board for a Group

  • Create a Board for a Training

  • Create a Board for a Training Path

  • Create a discussion or more in a board

Create a Board for a partner

To create a board associated to a partner, you must check the box "Create a board on the Community page" when creating your Partner.

All users linked to this Partner will have access to the relevant board on the Community page.

By default, only Administrators have the possibility to write messages on this board, but it is possible to enable this possibility for other users.


Create a Board for a Group

To create a board associated with a group, you must check the box "Create a board on the Community page" when creating your Group.

All users linked to this Group will have access to the relevant space in the Community.

Group managers, administrators, community managers and partner administrators can add discussions, delete a discussion and moderate messages if needed, on this board. Learners can only write a message inside a discussion.


Create a Board for a Training

To create a board associated with a training course, you must check the box "Create a board on the Community page" when creating your Training.

All users registered for this training and / or facilitators-designers/creators of this training will have access to the relevant space in the Community.

Designers/creators, moderators, administrators, community managers and partner administrators can add discussions, delete a discussion, moderate the messages, if needed, on this board. Learners can only write a message inside a discussion.


Create a Board for a Training Path

To create a board associated with a training path, you must check the box "Create a board on the Community page" when creating your training path.

All users registered for this training path will have access to the relevant space in the Community.

Creators/designers, moderators, administrators, community managers and partner administrators can add discussions, delete a discussion, moderate messages, if needed, on this board. Learners can only write a message inside a discussion.


Create a discussion

Administrators and Community Managers can create discussions within the various existing Boards.

To create a discussion, open the relevant board and click on Add a new talk.

You can create as many as you want.

All users who have access to the Board to which the discussion is attached will also have access to the discussion.
