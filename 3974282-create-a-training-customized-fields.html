Create a training > Customized fields

Last updated on 07/30/2020
Written by Claire Philippe
Updated over a week ago

When creating a training course, the following information must be completed:

  • General information

  • Detailed information (optional)

  • Configuration (optional)

  • Information about the organization (optional)

  • Customized fields (optional)

This article focuses on the Customized fields section.

This section will only appear if an Administrator (only Internal Administrators have permissions to do so) has previously created one or more customized fields related to Training.

Associating customized fields with Training can be useful for subsequent reporting purposes.

Customized fields are created during Pre-configuration (Settings ⚙️ > Customized fields).

See Artcile Settings - Customized fields.
