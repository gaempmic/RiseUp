Training Management

Last updated on 08/14/2020
Written by Kevin Legros
Updated over a week ago

Below is the Training Management Menu (access via Side Menu > Training)

View: Allows you to display the list of all training courses, regardless of their state (draft, awaiting validation, published). A sorting feature with several criteria makes it possible to quickly find the training sought, to modify it, delete it or duplicate it (it allows you to create a new training more quickly). This is where you can access a course in 'draft' state to continue working on it (that is say, edit it) and finalize it for publication. For more information see Manage trainings created.

Create: Allows you to create a new training course, whether it is an e-learning course, classroom training or "blended learning" training.

Validate trainings: Allows direct access to trainings that have been created by "Designers" to validate them and therefore publish them. In the most common configuration, designers can actually create training courses, but the publication of training courses requires the approval of an Administrator.

For more information see Manage trainings created by Designers.

Pending corrections: Allows direct access to exams and quizzes taken by learners to correct them.

Manage classroom sessions: Allows you to create classroom sessions, modify them, cancel them, validate them and also manage user registrations for the various classroom sessions.

Manage categories: Allows you to create training categories and subcategories and view them. When you create a course, you can attach it to one or more categories (assuming that these categories have been created beforehand). This makes it possible to help search for training in the same category, to carry out reports specific to one or more categories or to define objectives for users that are linked to a training category (e.g. spending x hours of training during the month of June on all training in the category 'mastery of safety rules').

⚠️ Only Administrators of the main Platform can create Training Categories. Partner Administrators do not have access to this section.
