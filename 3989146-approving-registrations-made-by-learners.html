Approving pending requests

  • Either the administrator (internal or partner, as applicable), the designer or even the Group Manager directly registers the user for training (by single user registration or using a linked group).
    This is what we have just seen in the previous stage.

  • Or User take the initiative by registering themselves for a training course accessible and visible in the Catalog. In this case, as part of the default configuration, this user registration is only a simple registration request that has to be approved administratively to trigger the actual user registration for training.
