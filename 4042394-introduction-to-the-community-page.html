Introduction to the Community page

How do I use the Community ? What is the objective of the Community ?
Written by Kevin Legros
Updated over a week ago

The Community page can be accessed by clicking on Community on the horizontal bar on top of the screen.

The Community page contains boards allowing people linked to those spaces to interact.

The community page includes the following elements :

  • The Main board

  • Group boards

  • Training boards

  • Training path boards


The Main board

It is made of different boards :

  • General : Accessible to all, it enables communication with all users, for example on new products, latest developments, the company's training policy etc.

  • Internal : Accessible to internal users, they are the only ones that can write on it.

  • Partner(s) : For each partner platform created, it is possible to configure a dedicated board bearing the name of this partner. It allows you to write to the users of the only partner concerned. To create a board associated with a partner, you must check the box "Create a board on the Community page" when creating your Partner.

Group, Training and Training paths boards

Administrators as well as Community Managers have access to all Boards on the Community page. Other users (learners, group managers, creators/designers, animators) have access only to the boards that concern them.

To create Group, Training and Path Boards, you must access the Group, Training or Training Path creation page concerned. When creating each of these elements, you can check "Create a board on the Community page ", and thus create a dedicated board.

For more details please see "how to create a board and a discussion in the community page".


Discussions within the Community page

Within each Board, it is possible to initiate discussions by creating specific spaces. Only administrators and Community Managers can create a new discussion inside a Board. All users concerned by the board can participate in the Discussion created.

User notification

When new messages are posted on the community, learners are notified by an icon indicating the number of unread available messages in their interface.

Notification on the vertical banner

Notification in Community dashboards

In addition, by default, if new messages are pending reading for the user, a daily email summarizing all the unread messages is sent.
