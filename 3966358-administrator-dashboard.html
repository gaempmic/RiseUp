Administrator Dashboard

Last updated on 06/17/2020
Written by Claire Philippe
Updated over a week ago

Below is a screenshot of an Administrator’s Dashboard.

Icons outlined in red are only visible to Internal Administrators (i.e. the Administrators of the Main Platform, known as the Internal Platform).

Below you will find the meaning of the main icons and buttons found on the platform:

  • Configuration

  • Hamburger Menu

  • View

  • Edit

  • Pin

  • More options

  • Search

  • Notifications

  • Helpcenter

  • Calendar

Configuration

This icon ⚙️ only appears in the screen-top banner for administrators of the platform and gives them access to the platform’s configuration.

When modifying any part of the platform’s configuration, don’t forget to save the changes made by going to the bottom of the page and clicking the Save button, generally at bottom right of the page. Otherwise, no modification will be implemented.

Hamburger menu

Also called Main management menu or dropdown menu.

It is at top left of the screen-top banner for administrators (of the main platform and, where applicable, partner platforms), designers, facilitators and group managers. Everyone will have access to a variety of sections of the dropdown menu, depending on their role.

View

This Eye icon lets you view the selected item (training, user, etc.). When clicking on the icon, you access to the related detailed information.

Edit

This pencil icon allows you to edit the element concerned (training, session, etc.). By clicking on the icon, you will access the element modification interface.

Pin

Clicking this button highlights the corresponding item by moving it to the top of the screen for greater visibility. Clicking it again will “un-pin” the item.

More options

Clicking this icon brings up a menu with several options appropriate to the context

Search

The magnifying glass at top right of the screen-top banner performs a quick search for particular training or a particular user (for administrators, designers, facilitators, etc.) and for learners a quick search for a particular training course.

Notifications

In My Notifications, users have access to all their notifications related to administering the Platform. For example, a training designer will receive notification when any training they have created has been approved. This icon is not visible to Learner profiles.

Help center

By clicking the Help Center, administrators have access to the Rise Up Help Center presenting all information and features on the platform.

Calendar

It is possible to go straight to your calendar to find the classroom sessions (possibly virtual classes) for which you are registered.
